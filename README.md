# Projeto TP3AI

## Descrição do Projeto
Este projeto é uma aplicação de Processamento de Linguagem Natural (PLN) que analisa sentenças utilizando uma gramática livre de contexto (CFG) e identifica chunks de frases nominais. A aplicação foi desenvolvida em Python utilizando a biblioteca NLTK.

## Arquivos
- **parser.py**: O script principal em Python que contém a implementação do analisador de sentenças.
- **requirements.txt**: Especifica as dependências necessárias para o projeto.
- **diretório sentences**: Contém arquivos `.txt` com sentenças para serem analisadas.
- **1.txt a 10.txt**: Arquivos de exemplo com sentenças para análise.
